"Display: inline" Vs "Display: inline-block"

The Battle of the Ages

10/18/2015

Oh, CSS, can't live with you, can't live without you. With the plethora of viewing options in the day and age, whether it be laptop, desktop, phone, or tablet, knowing how to display and format a website correctly is ever more important and ever more challenging... but fear not, for we'll cover a simple but important concept: 'display: inline' vs 'display: inline-block'.

By default, CSS is 'inline', meaning that if you add an element, it won't necessarily break-up the text or flow of elements.

This is an inline element. If I change the padding or the margins, it pushes things away horizontally, but not vertically.
