Enumerables and the Map Method

11/1/15

Enumerables are basically '.each' methods that work on different class types. They're 'mixins', meaning they're method blocks of code that can be mixed in with different classes. They iterate through a collection and execute the code you want it to on each element of a collection of data. So if you're creating an enumberable collection, also known as a module, you only need to define the methods once, define what they do for each class of object, and then that collection of methods is bundled together. Each class that has its own enumerable method in the module can use that enumerable.

First let's look at the map method. Let's say that we have an array of numbers:

nums = [21,19,37,55]

Now we want to alter each element in the collection by the same degree, such squaring everything. We can use this using the map method:

nums.map! { |element| element * 2 }

==> nums = [441, 361, 1369, 3025]
