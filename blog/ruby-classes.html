Ruby Classes

11/9/15

Since at least the time of Plato, philosophy has been concerned with the difference between existence and essence. The thing itself and the idea which informs the thing. Another way to think about this is the original definition of stereotype, it's like a printing plate from which you print out copies of something. If you have a triangle, there are certain things it must have, qualities that all triangles share: it must have 3 adjacent sides, 3 angles that add up to 180 degrees. Similarly, classes in ruby are for items that will share attributes and behaviors.

Suppose you own a car manufacturing factory. You want to make multiple copies of a car.

class Car

  def initialize(num_doors, engine, color)
    @num_doors = num_doors
    @engine = engine
    @color = color
    @engine_on = false
    @is_accelerating = false
  end

  def start_engine
    @engine_on = true
  end

  def accelerate
    @speed = 1
    @is_moving = true
    while @is_moving = true
      @speed += 1
    end
  end

  def decelerate
    while is_moving = true
      @speed -= 1
    end
    if @speed = 0
      is_moving = false
    end
  end

end

In the above example of code, for class car the first method is to initialize the object, like assembling a car in a factory line. When you create it, you want to decide what color it is, how many doors it'll have, and what kind of engine it'll have. And then we use instance variables (those variables with the '@' symbol) to pass values from method to method and also to store values for each car object. Each car object comes with those attributes, and it also comes with methods which can be thought of as car behaviors. All objects in this class, so all cars can accelerate, decelerate, or have their engines turned on. And the values of these attributes can be stored in the instance variables. Also, in this way those values can be passed from method to method. If the car is accelerating, we can increase the value of @speed. Similarly, we can give have the method decelerate modify that value and even bring the car to a stop.
