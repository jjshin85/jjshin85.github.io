Looping in Ruby and Looping in Javascript

11/17/15

There are a variety of ways to loop in Ruby, using the .each method, or the for or while loops. The use of these methods looks like this:

array = [1,1,1,1]
sum = 0
for i in 0...array.size
  sum += array[i]
end
#=> 4

In the above example, for i in 0...array.size tells ruby to iterate for variable i from 0 to 4, which is the size of the array. All done with a minimum of expressions. Also, the ... means to exclude the last digit, whereas ... would mean to include it within the range.

Javascript is a bit more cumbersome:

var array = [1,1,1,1];
var sum = 0; for (var i = 0; i < array.length; i++)
{
  sum += array[i];
}

After declaring variable i, you tell the program the range you want it to go until, which in this case is i < array.length. Then you declare how you want to iterate i, which in this case is by adding 1 to each iteration, which is represented by i++. And then all of these are separated by a semicolon. Also important to note is you must specify that each variable is a variable by putting var in front of the variable name every time you first declare it.

Ruby also has more built in methods that will allow you to loop through something, such as .each, or span class="block">.map.
